# MarketBot
Макет тг бота для покупки.<br>
Сделал сам за пару часов <b>(без гпт)</b>, чтобы чем то заполнить свое портфолио.<br>
Кому надо, могу [допилить за недорого)](https://t.me/clossedheimer)

## Установка
Сначала распакуйте файлы.<br>
Далее, настройте файл <b>config.py</b>, это несложно так как там все расписано.<br>

После, установите нужную библиотеку:
```bash
pip install aiogram
```

Теперь можете смело запускать <b>bot.py</b>:
```bash
python bot.py
```
